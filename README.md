#README
##Responde a la pregunta 1
- En produccion no es buena practica usar select * ya que traera todas las columnas de una tabla lo que genera mas demoras en la consulta y trae datos innecesarios para el analisis.
##Responde a la pregunta 2
- Una tabla puede tener nombre de columnas tecnicos, nombrarlas con alias facilita la lectura y comprension de los datos por parte de los stakeholder. Por ejemplo total_amount puede leerse como monto_total.
